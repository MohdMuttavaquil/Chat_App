# 💬 Msgly – Real-Time Chat Application

## 🌐 Live Demo
🚀 [Live Demo](https://chat-app-1-h1qb.onrender.com/)

## 📌 Description
Msgly is a full-stack real-time chat application that enables users to communicate instantly with a modern and responsive user interface. The app supports real-time messaging as well as offline message handling, ensuring users never miss important conversations.

Built with a scalable architecture, Msgly delivers fast, secure, and seamless communication across all devices.

---

## 🚀 Features

### 💬 Chat Features
- Real-time messaging using WebSockets  
- Offline message handling  
- Instant message delivery & updates  
- Clean and modern UI  
- Fully responsive (mobile, tablet, desktop)  

---

### 🔐 Authentication & Security
- JWT-based authentication  
- Secure environment variables using dotenv  
- Protected routes for authorized users  

---

## 🧑‍💻 Tech Stack

### Frontend
- React.js  
- Tailwind CSS  
- Axios (API communication)  

---

### Backend
- Node.js  
- Express.js  

---

### Real-Time Communication
- Socket.IO (WebSockets)  

---

### Database
- MongoDB (MongoDB Atlas)  
- Mongoose (schema modeling)  

---

### ☁️ Deployment & DevOps
- Frontend hosted on Vercel  
- Backend hosted on AWS EC2  
- Nginx for server configuration & SSL  
- CI/CD using GitHub Actions
